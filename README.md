exercise1.py: Reads JSON dataset and converts it into a proteinList of
proteinEntry objects using Pydantic. Functions include: calculating total 
protein mass, identifying large proteins based on sequence length >= 1000, and 
filtering out non-eukaryotic proteins. Answers are printed when the script is
ran.

exercise2.py: Reads JSON dataset and converts it into a CSV file with specific
columns.

proteins.csv: output of exercise2.py

exercise3.py: Reads JSON dataset and converts into XML format with one root
element and readability.

proteins.xml: output of exercise3.py

exercise4.py: Reads JSON dataset into YAML format with original data structure 
and ordering.

proteins.yaml: output of exercise4.py
